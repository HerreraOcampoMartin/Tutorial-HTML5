Más elementos HTML

Como prometí, aquí hay una lista de nuevas etiquetas para usar:

<a></a>

Es un texto el cual al hacerle clic, te lleva a otra página (un link).

Ir a Google 
							
<a href="https://www.google.com">Ir a Googles</a>

Como verás, esta etiqueta recibe un parámetro llamado “href” en el cual vas a poner el link al que quieras redireccionar al usuario.

<table></table>

Es un contenedor, en el cual se puede escribir contenido como si fuera una tabla. Quizás esta sea un poco complicada de ver al principio.

Hola Soy
Una Tabla 
								
<table>
  <tr>
	<td>Hola </td>
	<td>Soy </td>
  </tr>
  <tr>
	<td>Una </td>
	<td>Tabla </td>
  </tr>
</table>

Si es complicado ver la tabla, agrega en la primera etiqueta “<table>” un “border=1”, así:

								
<table border=1>

Pasando a la explicación, para cada fila horizontal, hay que poner una etiqueta “<tr></tr>” dentro de este “tr”, vas a poner un “<td></td>” el cual representa una celda, en donde se puede escribir. Si te lo estás preguntando, dentro de un “<td>” se puede poner cualquier otra etiqueta, como “<img>”, “<p>”, “<h1>”, e incluso otro “<table>”.

<ul></ul>

Es una lista como la que ves abajo.

  • Texto de ejemplo 1
  • Texto de ejemplo 2
								
<ul>
	<li>Texto de ejemplo 1</li>
	<li>Texto de ejemplo 2</li>
</ul>

Para indicar un ítem, coloca una etiqueta “<li></li>” y dentro escribe lo que quieras.

<ol></ol>

Es casi igual que la anterior con la diferencia de que en vez de puntos, hay números.

  1. Texto de ejemplo 3
  2. Texto de ejemplo 4
								
<ol>
	<li>Texto de ejemplo 3</li>
	<li>Texto de ejemplo 4</li>
</ol>

Para indicar un ítem, coloca una etiqueta “<li></li>” y dentro escribe lo que quieras.
