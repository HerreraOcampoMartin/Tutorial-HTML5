¿Cómo crear páginas web?

Seguro que todo el mundo alguna vez se ha preguntado, ¿Cómo se crean las páginas web? ¿Quién crea los sitios web? ¿Qué debo saber para hacer una web? Bien, existen muchas maneras de crear sitios web, existen herramientas que hacen la labor sencilla o se puede escribir código. Para resumirlo, te dejo una pequeña lista enumerando los distintos métodos que se usan para el desarrollo web:

La forma tradicional, escribir código

La primer forma de crear páginas web es escribiendo código, es la más complicada, pero te permite hacer todo lo que quieras si sabes lo que se hace. Tanto con el diseño, como con la programación se puede hacer casi todo lo que se te ocurra. El problema con esto es que para dominar el código hay que practicar y aprender ciertos conceptos. Sin embargo, no creas que esto lo puede hacer un grupo reducido de gente, cualquiera puede aprender, a cualquier edad. Las tecnologías más básicas para el desarrollo web son: HTML (para declarar la estructura de la página web), CSS (para darle un aspecto) y Javascript (para crear funciones y la parte lógica). Un CMS

¿Qué es un CMS? Proviene de las siglas de Content Management System (Sistema de Administración de Contenido) el cual permite la creación de páginas web sin escribir nada de código. El más conocido de todos es Wordpress, pero existen otros como Drupal, PrestaShop, etc. Esta herramienta es tan sencilla como editar la página web a tu antojo. También existen otros programas que son más sencillos de utilizar, el que más recomiendo es Google Sites, ya que lo único que necesitas para usarlo es una cuenta de Gmail.

¿Usar un CMS o escribir código?

Depende de lo que quieras hacer, por ejemplo, para tu blog es mucho mejor usar un CMS u otra cosa que te genere un sitio web como Google Sites, ya que tiene todo lo que necesites y es más rápido y fácil de usar. Pero para aplicaciones web, es decir, programas más complejos deberías escribir código, por ejemplo Facebook, Youtube, E-mails o la mayoría de cosas que tengan más funciones que simplemente texto, es obligatorio usar código, un CMS no puede crear eso. Sin embargo, hay gente que se dedica a crear plugins para CMS que te provean ciertas tareas (me refiero a una sección de comentarios comentarios, botones que cumplan ciertas acciones, sistemas de usuarios básico, no necesitas programar nada) aunque la mayoría de los casos, si vas a hacer aplicaciones grandes se necesita un equipo de programadores, diseñadores, etc.
