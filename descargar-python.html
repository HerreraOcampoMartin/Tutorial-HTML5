Como descargar e instalar python

Hola, bienvenido a este artículo en el cual podrás aprender a descargar e instalar Python. En el artículo anterior te hice un breve resumen de qué es Python y para que se usa.

Sin más, empecemos Primero, ir a al link de descarga de Python, este link te llevará automáticamente a la última versión del lenguaje. Clic aquí Al darle clic al botón amarillo que dice “Download Python”, empezará tu descarga. Por defecto debería darte la opción de descargar Python para tu sistema operativo, pero si ese no es el caso, justo abajo del botón amarillo están los diferentes sistemas operativos

Luego, vas a ir a descargas (o la carpeta donde hayas guardado el archivo). Al darle doble clic, se abrirá una ventana de instalación, el proceso es muy sencillo, es principalmente darle a “siguiente”, este proceso de instalación es para configurar el lenguaje, pero las opciones por defecto son las mejores.

Por último para verificar que hayas instalado correctamente Python, ve al buscador de windows y escribe “cmd”, el cual te sugerirá una aplicación que es la terminal de tu computadora. Cuando se abra, vas a ver una ventana en negro con un cursor y vas a escribir: “python”. Si aparece un mensaje indicando una versión de Python y debajo “>>>” con otro cursor ¡entonces has instalado Python correctamente!

Actualmente estás en el modo de escritura de código, para salir solo escribe “exit()”
