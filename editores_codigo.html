Editores de código

Bienvenido nuevamente a este pequeño curso de introducción al desarrollo web. Anteriormente, hemos estado haciendo una pasada por arriba de HTML. Hoy, vamos hablar sobre los editores de código. ¿Qué es un editor de código? Es un programa el cual nos ayudará a escribir código más fácil y rápidamente. Si has estado usando el bloc de notas, o codepen, sabrás que hay tareas un tanto pesadas y repetitivas, como escribir la estructura de un archivo HTML. Un editor de código puede completar eso por nosotros, mostrar sugerencias, marcar errores y mucho más. Existen bastantes editores de código, aquí una lista de los más conocidos:

  • Visual Studio Code
  • Sublime Text
  • Atom
  • Vim

Todos cumplen con la función de un editor de código, y son excelentes. En mi caso, yo uso Visual Studio Code (abreviado VS Code), ya que tiene muchas extensiones (funciones creadas por otras personas para complementar al programa), es ligero, simple y es desarrollado por Microsoft

¿Cómo instalo VS Code?

Bien, primero, hay que ir a la página oficial de VS Code, Busca un botón azul que dice “download” o “descargar”. Luego, es tan sencillo como descargar el programa, hacer doble clic en el instalador, el proceso de configuración es muy intuitivo (es prácticamente darle a “aceptar” y “siguiente”, aunque te da la opción de configurar ciertos aspectos. He de remarcar que el programa está disponible para Windows, Mac y Linux

Una vez que hayas instalado y abierto VS Code, ve a cualquier carpeta en la que te gustaría guardar los archivos de tu página web (preferiblemente, que esté vacía) y arrastrala hasta VS Code, o haz clic derecho sobre la barra en la izquierda de VS Code, y agregar nueva carpeta. Cuando se abra la carpeta en la barra de menú a la izquierda, haz clic derecho sobre el nombre de la carpeta > “Nuevo archivo” y escribe “index.html”. Dentro del archivo escribe “!” y unos segundos después te saldrá un cartel que dice nuevamente, “!”, presiona enter y ya está la estructura de HTML básica. Si no te sale el cartel, presiona a la vez Ctrl + Espacio y espera unos segundos.
