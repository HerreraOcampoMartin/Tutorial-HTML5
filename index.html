Introducción a HTML

HTML es un lenguaje de marcado ¿Y qué es eso? Es un código que escribimos para diseñar la estructura de una web. La idea, es enseñarte a usar este lenguaje para crear páginas web.

¿Por dónde empiezo?

Para empezar, vas a necesitar un lugar donde escribir tu código, se puede usar una página web,un programa o mismamente, el bloc de notas. Para tu primer código da igual que uses, pero cuando pase más tiempo, es recomendable que uses un programa, llamado editor de código, ya que brinda muchas facilidades a la hora de trabajar, personalmente uso Visual Studio Code, fue creado por Microsoft. Si vas a seguir este curso, es suficiente con usar una página web llamada codepen quepermite escribir código sin descargar nada.

Sin embargo, hoy podrás empezar con el bloc de notas. Primero, abre el bloc de notas y escribe en él: 

							
						< h1 > Hola mundo < /h1 >
Luego, dale a archivo -> guardar como -> y en el nombre del archivo escribe:
primer página.html
Es muy importante que termine en ".html", si termina en ".txt" u otro formato, no funcionará. Una vez hecho esto, abre el archivo que acabas de guardar, dale doble clic, y debería abrirse tu navegador con el mensaje que hayas puesto entre los "< h1 >". Si es así ¡Felicidades, esa es tu primera página web!

Hola mundo
